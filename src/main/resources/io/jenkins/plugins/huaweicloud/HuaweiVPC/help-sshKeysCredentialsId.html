A credential of the type "SSH Username with private key" with the RSA private key of the ECS key pair. You can use the Huawei Cloud Console or any other tool of your choice to generate the key pair. The username is used, and the credential should not have a passphrase.

An RSA private key for EC2 is a textthat starts with "-----BEGIN RSA PRIVATE KEY-----".
recommended to set the certificate ID to the name of the created key on HUAWEI CLOUD Console.
